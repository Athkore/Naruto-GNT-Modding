# Replacing Character Select Screen Animations

Animations in the character select screen can be replaced using the [replace_anim.py](/utils/replace_anim.py) script.

All of the animations loop, however some move the character during the animation. Therefore some will result in the character eventually moving off-screen.

Here is a collection of animations I thought worked decently as character select screen animation replacements.

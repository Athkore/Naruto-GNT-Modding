# Opcode Group 00

## 0000 - soft_reset

Terminate the program counter.

- Parameters: 0
- Terminate: Always

## 0001 - hard_reset

Terminate the program counter and `stored_pc`.

- Parameters: 0
- Terminate: Always

## 0002

- Parameters: 0
- Terminate: Always

## 0003 - debug00

Seems to be debugging related. Calls sprintf which prints something like `sys_bp(): seq_p802c6718 reg_p802d7760 pc802c6790`.

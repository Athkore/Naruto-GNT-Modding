# Opcode Group 00

## 0000

Terminate the program counter.

- Parameters: 0
- Terminate: Always

## 0001

Terminate the program counter and set `param_2[0x16] = 0`

- Parameters: 0
- Terminate: Always

## 0002

- Parameters: 0
- Terminate: Always

## 0003

Seems to be debugging related. Calls sprintf which prints something like `sys_bp(): seq_p802c6718 reg_p802d7760 pc802c6790`.
